# Sistema de Estoque em Python

Esse é um projeto que eu desenvolvi enquanto aprendia programação com Python. A ideia foi criar um sistema simples, mas funcional, que simulasse uma loja com cadastro de usuários, controle de produtos e compras.

Durante o desenvolvimento, fui aprendendo na prática como organizar melhor o código, lidar com erros e estruturar um sistema mais completo.

---

## O que o sistema faz

* Cadastro e login de usuários
* Diferenciação entre **admin** e **cliente**
* Cadastro, remoção e visualização de produtos
* Controle de estoque
* Carrinho de compras
* Cálculo do valor total da compra
* Simulação de pagamento (cartão, boleto e pix)

---

## O que eu pratiquei nesse projeto

* Lógica de programação
* Uso de listas e dicionários
* Funções e organização em múltiplos arquivos
* Leitura e escrita de arquivos JSON
* Tratamento básico de erros

---

## Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/clebersondev-py/sistema_estoque
```

2. Entre na pasta:

```bash
cd sistema_estoque
```

3. Execute:

```bash
python main.py
```

---

## Observações

* O sistema roda no terminal
* Os dados são salvos em arquivos `.json`
* Caso os arquivos não existam, o sistema cria automaticamente
